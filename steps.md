# Roadmap: Разработка лендинга «Очень вкусные дела»

Этот план предназначен для пошаговой реализации проекта. 
**Важно:** выполняй задачи строго по порядку. После каждого этапа проверяй отсутствие ошибок в терминале и корректность отображения в браузере.

---

## Этап 1: Окружение и Базовая конфигурация

1. **Инициализация:** 
   - Создай проект на **Astro 5.x** (используй TypeScript, пустой шаблон).
2. **Tailwind:** 
   - Установи и настрой Tailwind CSS:
     ```bash
     npx astro add tailwind
     ```
3. **Конфигурация темы:** 
   - В `tailwind.config.mjs` добавь палитру на основе логотипа:
     - `brand-pink`: `#EB00FF` (основной неон).
     - `brand-dark`: `#0A0A0A` (фон).
   - Настрой кастомный `box-shadow` для эффекта свечения (`glow`).
4. **Глобальные стили:** 
   - В `src/styles/global.css` установи темный фон и шрифты.
   - Проверь, что `npm run dev` запускается без ошибок.

## Этап 2: Подготовка контента (Data Layer)

1. **Структура данных:** 
   - Создай файл `src/data/menu.json`.
2. **Наполнение:** 
   - Опиши минимум **8 позиций завтраков**. Каждому объекту добавь:
     - `id`, `title`, `description`, `price`.
     - `image` (путь к локальному файлу).
     - `bentoSize`: значение для сетки (`small`, `medium`, `large`).
3. **Ассеты:** 
   - Создай папку `src/assets/images/` и добавь туда заглушки или реальные фото блюд.

## Этап 3: Верхний блок (Hero Section)

1. **Компонент заголовка:** 
   - Создай секцию с названием «Очень вкусные дела».
2. **Стилизация:** 
   - Примени к тексту неоновое свечение (используй `text-shadow` или фильтры Tailwind).
3. **Подзаголовок:** 
   - Добавь текст «Меню завтраков» и дескриптор заведения.
   - Убедись, что на мобильных устройствах текст не обрезается.

## Этап 4: Компонент карточки (The Menu Card)

1. **Создание компонента:** 
   - Разработай `src/components/MenuCard.astro`.
2. **Верстка карточки:**
   - Используй `astro:assets` для автоматической оптимизации.
   - Добавь оверлей с градиентом, чтобы текст описания легко читался поверх фото.
   - Реализуй **Glassmorphism-эффект** для ценника.
3. **Интерактив:** 
   - Добавь `hover`-анимацию (мягкое увеличение карточки и усиление свечения рамки).

## Этап 5: Сборка Bento-сетки

1. **Логика сетки:** 
   - На главной странице реализуй контейнер `grid` с классами `grid-cols-1 md:grid-cols-4`.
2. **Маппинг данных:** 
   - Настрой вывод карточек из JSON.
3. **Bento-раскладка:** 
   - Напиши вспомогательный класс или конструкцию `switch/case`, которая будет присваивать нужные `col-span` и `row-span` в зависимости от поля `bentoSize` из данных.
4. **Проверка:** 
   - Убедись, что сетка выглядит сбалансированно и адаптивно.

## Этап 6: Оптимизация и Lightbox

1. **Lazy Loading:** 
   - Проверь, что у всех изображений в сетке стоит атрибут `loading="lazy"`.
2. **Lightbox:** 
   - Реализуй модальное окно для просмотра фото.
   - При клике на карточку фото должно открываться на весь экран.
   - Используй библиотеку (например, `fslightbox`) или напиши минимальный скрипт на ванильном JS.
3. **Блокировка скролла:** 
   - Убедись, что при открытом фото основной контент не прокручивается.

## Этап 7: Финализация и Деплой

1. **Анимации:** 
   - Добавь плавное появление карточек при первой загрузке (через CSS `@keyframes` или `framer-motion`).
2. **SEO:** 
   - Добавь базовые мета-теги и favicon.
3. **Production Build:** 
   - Выполни команду:
     ```bash
     npm run build
     ```
   - Проверь лог на наличие предупреждений об оптимизации картинок.
   - Исправь ошибки TypeScript, если они возникнут.
